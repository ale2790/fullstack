1- GIT INIT: inicia un repositorio local (copia todo lo que tengo en el directorio donde la inicio)

2- U significa sin seguimiento: no estan guardados en el repositorio local.

3- GIT ADD: agrega todos los datos al repositorio local.

 -GIT ADD index.html : agregar e archivo al repositorio local, agregando el nombre del archivo.

 - git add . : agrega todo el directorio donde estamos trabajando.

4- A: archivo agregado al repositorio local.

5- M: el archivo fue modificado, pero no guardado.

6- GIT COMMIT : embalar el paquete de todo el archivo para enviarlo.
  -m : comentar o describir la tarea realizada.
  
7- Si tocamos algo, volvemos a iniciar, menos el numero 1.

8- GIT STATUS: consulta el estado de GIT.

9- GIT No guarda directorios vacios por default.

10- GIT REMOTE -v:
